# Custom nodes RasterRelay

To miejsce jest przygotowane pod przyszłe własne custom nodes dla ComfyUI.

Custom nodes mogą później pomagać w komunikacji między:

- Photoshopem,
- Launcherem,
- ComfyUI,
- workflow inpaintingu.

Na pierwszym etapie nie tworzymy jeszcze paczki node'ów. Przygotowujemy tylko czytelne miejsce w repozytorium.
